# Jenkins Installation Script

## Overview
This script automates the installation and configuration of Jenkins on Ubuntu/Debian systems.

## Prerequisites
- Ubuntu 20.04/22.04 or Debian 11/12
- Sudo privileges
- Internet connection

## Installation
```bash
# Make the script executable
chmod +x install-jenkins.sh

# Run the script
sudo ./install-jenkins.sh









Features
Installs Jenkins LTS version

Configures Jenkins as a service

Opens required firewall ports

Optional plugin installation

Configuration
Edit the script variables at the top:

JENKINS_VERSION: Jenkins version to install

JENKINS_PORT: Port for Jenkins web interface
